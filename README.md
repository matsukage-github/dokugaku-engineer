## 内容
PHP、データベース、Dockerなどを動画で学び、簡単なWebサービスを開発する

### カリキュラム
WEB系のサーバーサイドエンジニアとして実務で自走できる状態を目指す。
エンジニアとして就職・転職できるようになるのはもちろん、その後の実務で、未知の課題に出会っても自分で情報収集し解決できる実践的スキルを一緒に身に付ける。
